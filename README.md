## Add a dotfile to be tracked 

To add a new config file directory :
``` bash
tuckr push groupname path-of-files-or-folder
```

## To update a group 

The first time get the files as they are :  
```shell
tuckr add -a groupname
```

Later on do :
```shell
tuckr add groupname
